Actividad 1 (Valor 10 puntos)
Investiga en diferentes sitios web y describe cada tema:

  1. Definición de CSS:CSS (Cascading Style Sheets) es un lenguaje utilizado para describir la apariencia y el formato de un documento escrito en HTML. Controla el diseño visual de páginas web.
  2. Funcionamiento Básico de CSS:CSS funciona asignando estilos (como colores, fuentes, márgenes, etc.) a elementos HTML mediante reglas que se aplican a través de selectores.
  3. Incluir CSS en un archivo HTML:
    • En línea:usando el atributo style dentro de una etiqueta HTML.
    • Interno:dentro de una etiqueta style en la sección head.
    • Externo:vinculando un archivo .css externo usando link en el head.
  4. Definición de una regla CSS: Una regla CSS define cómo se debe estilizar un elemento específico o un grupo de elementos en una página web.
  5. Elementos de una regla CSS:
    • Regla:El conjunto completo que aplica un estilo a un elemento.
    • Selector:Indica a qué elemento(s) HTML se aplica el estilo.
    • Bloque de declaración:Contiene una o más declaraciones, encerradas entre {}.
    • Propiedad:El aspecto del estilo que se quiere modificar (como color, font-size).
    • Valor:El valor asignado a la propiedad (como red, 16px).
  6. Medios CSS:Son reglas que se aplican solo bajo ciertas condiciones, como el tamaño de la pantalla, utilizando @media.
  7. Comentarios CSS:Se crean usando /* comentario */ y son ignorados por los navegadores.
  8. Sintaxis de las propiedades CSS:La sintaxis general es propiedad: valor;, por ejemplo, color: blue;.
